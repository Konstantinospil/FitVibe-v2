# ADR-010: Internationalization (Static + AI Hybrid)
**Date:** 2025-10-07  
**Status:** Proposed  
**Cross-References:** Chat 2025-10-06 / PRD §6  

## Decision

## Consequences
✅ Dynamic multilingual support  
⚠️ Requires moderation and API rate control
