# ADR-009: Accessibility & WCAG 2.1 AA Compliance
**Date:** 2025-10-07  
**Status:** Accepted  
**Cross-References:** PRD §11 / QA §10  

## Decision
Frontend must meet WCAG 2.1 AA: semantic HTML, contrast ratios, keyboard nav, ARIA labels.

## Consequences
✅ Inclusive UX  
⚠️ Slightly longer dev cycle
